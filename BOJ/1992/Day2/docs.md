### 백준 뒤집기 문제 (1992)

- 좀 더 재귀를 잘 활용하는 방법을 구사

- 분할정복 방식

  - 재귀를 통해서 Size 가 1일 때 까지 탐색
  - return 값을 비교하여 정복 함으로써 결과값 도출
    - r1, r2, r3, r4 값 비교
    - 다를 경우 ()로 묶어서 계산

- 어려웠던 점

  - r1 r2 r3 r4를 비교 할 때 r1 === r2 === r3가 아니라 r1 === r2 && r1 === r3로 해야 됨 (js에서)

  - 반례) r1 ,r2 값 비교 시 r1.length === 1을 추가해주자!! 정복 과정에서 () 가 추가된 값과 비교할 수도 있기 때문!!
